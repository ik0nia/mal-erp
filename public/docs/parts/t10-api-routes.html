POST /chat/message

ControllerChatController@message
Rate limitingthrottle:chat-message — 20 req/min per IP
AutentificarePublică (fără auth); CORS restricționat

Response JSON 200:

{
  "reply": "string — răspunsul AI",
  "session_id": "string (UUID)",
  "products": [
    {
      "id": 123,
      "name": "string",
      "price": "string",
      "image_url": "string|null",
      "url": "string — permalink WooCommerce",
      "stock_available": true,
      "unit": "string|null"
    }
  ],
  "contact_form": false,
  "contact_form_message": "string"

Comportament: Istoricul conversației în Cache (TTL 15 min, max 20 mesaje). Motor AI: Claude claude-haiku-4-5 cu Tool Use (searchProducts, checkQuantityAvailability, getOrderStatus, getCompanyInfo). Toate mesajele logate în chat_logs cu IP și tokeni AI. La eroare internă: mesaj generic fără a expune detalii tehnice.

4. Plugin WordPress Bridge Endpoints

Expuse de plugin-ul malinco-erp-bridge (v1.1.0) pe https://malinco.ro/wp-json/malinco-erp/v1/.
Autentificare: header X-ERP-Api-Key: {cheie} (cheie criptată în AppSetting::KEY_WOO_PLUGIN_API_KEY).
Client PHP: app/Services/WooCommerce/WooPluginClient.php — timeout 15s (60s pentru self-update).

Method Endpoint Request Response Descriere
GET /version — {version, status, site} Verifică dacă plugin-ul e activ și returnează versiunea curentă
GET /product/meta Query: woo_id sau sku Meta-datele produsului Citește meta-datele custom ale unui produs WooCommerce
POST /product/meta {woo_id: int, sku: string, meta: {cheie: valoare}} {success: bool} Actualizează meta-datele unui produs (ex: _furnizor_nume, _furnizor_sku)
POST /product/bulk-meta {items: [{woo_id, sku, meta}]} {results: {updated, failed, errors}} Actualizează meta-datele pentru mai multe produse într-un singur request
POST /product/recalculate {product_ids: [int]} {success: bool} Declanșează recalcularea disponibilității/prețului (plugin malinco-availability-pricing)
GET /options/{key} — Valoarea opțiunii WordPress Citește o opțiune WordPress via get_option()
POST /cache/flush — {success: bool} Golește cache-ul WordPress/WooCommerce
POST /self-update multipart/form-data: fișier ZIP (plugin) {success: bool} Self-update plugin: instalează automat noua versiune din ZIP-ul primit

Deploy plugin: php artisan woo:deploy-plugin construiește ZIP-ul din wordpress-plugin/malinco-erp-bridge/ și îl trimite via /self-update.
